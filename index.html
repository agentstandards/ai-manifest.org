AI Manifest（社区草案 v0.1）

/.well‑known/ai.json OpenAPI JSON Schema JOSE/JWKS

Why 映射 Best Practices PR文案

定义：AI Manifest 是放在域名 /.well-known/ai.json 的机器可读文件，用统一位置与格式暴露：能力清单（OpenAPI/JSON Schema 映射）、接入规则（端点/鉴权）、信任线索（JWKS/撤销）。

为什么作为“可选参考”

  • 可发现：沿用 .well‑known 约定。
  • 可互通：字段对齐 OpenAPI/JSON Schema，兼容 MCP 与社区 agents.json。
  • 可验证：指向 JWKS，支撑签名/轮换/撤销。

Best Practices

  • 路径：/.well‑known/ai.json；Content‑Type: application/json；GET/HEAD。
  • 版本：manifest_version（如 "0.1"）；ETag/Last‑Modified；max‑age=3600。
  • URL：绝对 HTTPS（OpenAPI/Schema/CRL/JWKS）。
  • 信任：auth.jwks_uri；kid；轮换窗口≥7天；撤销记时。
  • robots：允许抓取 /.well‑known/ai.json；敏感页 noindex。
